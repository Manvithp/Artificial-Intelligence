# Housing Price Prediction

## Problem Statement
The objective of this project is to develop a Machine Learning model that predicts house prices based on various property features such as area, number of bedrooms, bathrooms, and other relevant factors. The model analyzes historical housing data to learn patterns and provide accurate price predictions.

---

## Project Objective
- Understand the relationship between house features and price
- Perform data preprocessing and cleaning
- Train a regression model for price prediction
- Evaluate model performance using appropriate metrics

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Workflow
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Model Training (Linear Regression)
6. Model Evaluation
7. Price Prediction
